# 🍽️ Restaurant Management System (C Language)

A console-based Restaurant Management System developed in C that allows customers to browse menu categories, place food orders, calculate bills automatically, and generate the final invoice with GST.

---

## 📌 Project Overview

The Restaurant Management System is a menu-driven application built using the C programming language. It simulates a restaurant ordering system where users can:

- Browse different food categories
- Select food items
- Enter quantity
- Place multiple orders
- Calculate total bill with GST
- Display final invoice

This project demonstrates the use of functions, loops, conditional statements, switch-case, user-defined functions, and modular programming in C.

## 📖 How It Works

### Step 1
Run the program.

### Step 2
Select a food category.

Example:

1. Starters
2. Soup
3. Vegetables
4. Tandoori Roti
5. Rice
6. Dal

### Step 3
Choose the food item.

### Step 4
Enter quantity.

### Step 5
Choose whether to order more food.

### Step 6
View the final bill with GST.
